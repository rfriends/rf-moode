rfriends_moodeはmoode player環境でrfriends3を動作させるスクリプトです。  
  
cd ~/  
sudo apt -y install git  
git clone https://github.com/rfriends/rfriends_moode.git  
cd rfriends_moode  
sh rfriends3_moode.sh
